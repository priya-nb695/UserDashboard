# User Dashboard - Async JavaScript Practice

A simple User Dashboard built using **HTML, CSS, and JavaScript** to deeply understand:

- Callbacks
- Promises
- Async / Await
- Loading states
- Simulated API calls

---

## 📌 Project Goal

This project is built to understand how asynchronous JavaScript works by:

1. Simulating API calls using `setTimeout`
2. Loading user data
3. Loading user posts
4. Loading comments
5. Displaying everything in the UI

The same logic will be implemented in:

- ✅ Callback version
- ✅ Promise version
- ✅ Async/Await version

---

## 🖥️ UI Features

- Load User button
- Loading spinner
- User profile section
- Posts section
- Comments list

---

## 🧠 What I Learned

- How callback nesting works (callback hell)
- How Promises improve readability
- How async/await simplifies async code
- How the event loop handles async operations
- Difference between synchronous and asynchronous execution
